# نظام إدارة مواعيد العيادة الطبية

## وصف المشروع
نظام متكامل لإدارة مواعيد العيادة الطبية مبني باستخدام Angular 17. يوفر النظام واجهة سهلة الاستخدام للأطباء لإدارة مواعيدهم وتنظيم جداولهم اليومية.

## التقنيات المستخدمة
- **Angular 17** - إطار العمل الرئيسي
- **TypeScript** - لغة البرمجة
- **SCSS** - تنسيقات متقدمة
- **FontAwesome** - أيقونات
- **Bootstrap** - إطار عمل CSS

## كيفية التشغيل

### المتطلبات
- Node.js (الإصدار 18 أو أحدث)
- npm أو yarn

### خطوات التثبيت
```bash
# استنساخ المشروع
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
cd DoctorProject

# تثبيت التبعيات
npm install

# تشغيل المشروع
ng serve

# فتح المتصفح على
http://localhost:4200
```

## هيكل المشروع
```
src/
├── app/
│   ├── features/
│   │   ├── auth/          # نظام المصادقة
│   │   ├── doctor/        # واجهة الطبيب
│   │   └── Admin/         # واجهة المدير
│   ├── layouts/           # تخطيطات الصفحات
│   └── guards/            # حماية الصفحات
├── components/            # المكونات المشتركة
└── environments/          # إعدادات البيئة
```
